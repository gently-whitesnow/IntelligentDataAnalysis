# ЛР 3 — классификация изображений цветов

## Задание
Сверточная сеть и трансферное обучение для классификации изображений цветов из набора данных,
разнесенного по папкам классов.

## Данные
Подготовьте папку с изображениями, где каждый класс находится в отдельной директории:

```
data/
  rose/
  daisy/
  tulip/
  ...
```

Скрипт автоматически сформирует выборки `train/val/test` (по умолчанию 70/15/15) в каталоге
`splits/` рядом с исходными данными.

## Запуск
```bash
python lab_3.py --data /path/to/data --artifacts artifacts
```

## Результаты
В папке `artifacts` сохраняются:
- графики точности/потерь для двух моделей;
- метрики тестирования и сравнение моделей (`results.json`).
